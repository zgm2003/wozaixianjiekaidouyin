- 林夜、天机一型手机、使用说明玉简沿用既有资产；鬼财神、金不换以 `剧本资产/角色资产重制提示词.md` 生成后的新版资产为准。

- 按每 5 条约 15 秒复制“逐条视频文本”，逐段生成视频。
- 外部剪辑工具负责最终成片合成。
