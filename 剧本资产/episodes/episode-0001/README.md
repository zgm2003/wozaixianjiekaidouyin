## 使用顺序

1. 先用 `## GPT-image-2 参考图生成提示词` 生成或确认 6 个第1章参考资产：
   - 林夜三视图
   - 白清玄三视图
   - 魔门汇报三人组设定图
   - 鬼王宗宗门大殿内景
   - 神级文娱系统界面
   - 天机一型手机单体道具
2. 从 `## 每5条复制制作块` 开始，按组复制 5 条视频文本、上传参考图、音色和统一要求。
3. 每组里的 `上传参考图：资产名 = 图片N` 就是本组绑定说明。
4. 外部视频工具生成片段，外部剪辑工具负责合成最终成片。

## 验收重点

- 第1集只覆盖第1章，不进入第2章。
- 正片从鬼王宗宗门大殿开场，不再跳过前 5 组。
- 事件顺序保持：魔门战报 -> 林夜压住开战 -> 独处破防 -> 系统绑定 -> 文娱商城 -> 抖音纯享版和天机一型手机 -> 手机亮屏。
- 林夜必须是外冷内慌的正道卧底，不是主动嗜杀反派。
- 台词可以为了视频呼吸感轻微调整，但必须贴合原著事实，不能改变人物动机、因果和章节钩子。
- 系统必须是文娱/互联网系统，不是战斗外挂。
- 手机第一次亮屏必须是本集核心视觉爆点。
- 场景只使用鬼王宗宗门大殿内景，不出现奼女教山门、奼女教宗门大殿、鬼王宗大殿外石阶或极北荒地上空。
- 本集结尾停在第一部天机一型手机屏幕亮起音符图标，给第2章抖音空白页、自拍视频和招募女主播留下空间。
