# 第1集交付说明

## 文件

- `seedance-all-reference-feed.md`

## 覆盖范围

- 原文第 1 章：神级文娱系统

## 交付类型

ChatGPT-ready / Seedance 全能参考投喂包。

画幅：16:9。

本目录不包含图片文件，不包含视频文件，不包含 Canvas 包。

## 使用顺序

1. 先把 `seedance-all-reference-feed.md` 交给 ChatGPT，按“原著守则”和“镜头语言规则”复核。
2. 用 feed 中的 GPT-image-2 提示词生成或确认 6 个第1章参考资产，后续集数优先沿用这些已锁定资产：
   - 林夜三视图
   - 白清玄三视图
   - 魔门汇报三人组设定图
   - 天机一型手机单体道具
   - 鬼王宗宗门大殿内景
   - 神级文娱系统界面
3. 把参考资产按“参考资产绑定”上传到外部视频工具。
4. 按每 5 条约 15 秒复制“逐条视频文本”，逐段生成视频。
5. 外部剪辑工具负责合成最终成片。

## 验收重点

- 第1集只覆盖第1章，不进入第2章。
- 林夜必须是外冷内慌，不是主动嗜杀反派。
- 台词可以压缩原文，但必须服务剧情，不能机械照抄怪味句，也不能凭空改剧情。
- 系统必须是文娱/互联网系统，不是战斗外挂。
- 手机第一次亮屏必须是本集核心视觉爆点。
- 场景只使用鬼王宗宗门大殿内景，不出现奼女教山门、奼女教宗门大殿、鬼王宗大殿外石阶或极北荒地上空。
- 本集结尾停在第一部天机一型手机屏幕亮起音符图标，给第2章抖音空白、自拍视频和招募女主播留下空间。
