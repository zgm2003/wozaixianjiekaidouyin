## 交付类型

ChatGPT-ready / Seedance 全能参考投喂包。

画幅：16:9。

本目录只包含文本交付文件，不包含视频成片。图片素材统一保存到项目资产目录，并通过 Git LFS 上传云端管理。

## 使用顺序

- 先把 `seedance-all-reference-feed.md` 交给 ChatGPT，按“原著守则”和“镜头语言规则”复核。
- 按 feed 的“参考资产绑定”准备 10 个参考位。
- 林夜和天机一型手机使用既有云端资产，不重新生成。
- 新生成火龙真人、火云洞自拍场景、创作激励计划详情、抖+1000灵石投放、点赞评论粉丝暴涨、抖音官方直播间、火龙真人进入直播间被欢迎、抖音平台功能示意。
- 把新图片保存到 `人设资产/`、`场景资产/`、`道具资产/` 中对应目录，再 `git add`，由 Git LFS 管理并上传云端。
- 按每 5 条约 15 秒复制“逐条视频文本”，逐段生成视频。
- 外部剪辑工具负责最终成片合成。

## 验收重点

- 第 10 集 80 条，原因是创作激励计划、抖+投放和官方直播课堂解释性内容多，需要给视频呼吸感。
- 每 5 条非打斗段落对白/系统/旁白密度不能超过 3 条。
- 火龙真人不是纯小丑，他的核心是孤僻炼器大师被点赞、评论和欢迎弹幕击中。
- 创作激励计划、抖+1000灵石投放、点赞评论粉丝上涨、直播弹幕、平台功能、1000粉丝卖货门槛和10万在线必须清楚。
- 结尾停在直播间在线突破10万，不进入第11章随机连麦和后续手机售罄。
- 所有视频文本开头都有时间、内外和具体地点。
- 所有新图片提示词必须是 16:9，并以“4K画质！”结尾。
