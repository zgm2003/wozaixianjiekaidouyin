## 使用顺序

- 先把 `seedance-all-reference-feed.md` 交给 ChatGPT，按“原著守则”和“镜头语言规则”复核。
- 按 feed 的“参考资产绑定”准备 13 个参考位。
- 楚风、飞霞岛弟子、四海阁手机销售内景、天机一型手机、林夜使用既有云端资产；金不换、鬼财神以 `剧本资产/角色资产重制提示词.md` 生成后的新版资产为准。
- 新生成四海阁围观抢购、四海阁售罄关门、金不换三楼视频汇报、十路运财使售罄汇报、林夜灵石转账与系统商城补货、海上仙山静室再看20分钟。
- 把新图片保存到 `人设资产/`、`场景资产/`、`道具资产/` 中对应目录，再 `git add`，由 Git LFS 管理并上传云端。

- 外部剪辑工具负责最终成片合成。
